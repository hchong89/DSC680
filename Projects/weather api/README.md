# Weather API Application

## Overview
This project is a Python-based weather application that retrieves real-time weather information using the OpenWeatherMap API.

Users can search weather conditions by:
- ZIP code
- City and state

The application retrieves location coordinates, connects to the weather API, and displays current weather conditions using different temperature measurement systems.

---

## Technologies Used
- Python
- REST API
- OpenWeatherMap API
- JSON
- Requests Library

---

## Features
- Search weather by ZIP code
- Search weather by city and state
- Supports:
  - Fahrenheit
  - Celsius
  - Kelvin
- Displays:
  - Current temperature
  - Feels-like temperature
  - High and low temperatures
  - Pressure
  - Humidity
  - Weather description
