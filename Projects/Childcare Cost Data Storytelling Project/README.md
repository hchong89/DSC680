# Childcare Cost Data Storytelling Project

## Overview
This project analyzes childcare cost data across the United States to explore affordability trends and the financial impact of childcare on families. The project focuses on transforming complex cost data into meaningful visual stories using data visualization and storytelling techniques.

The analysis highlights regional differences in childcare expenses and examines how childcare affordability varies across states and household income levels.

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

---

## Data Sources
The project uses childcare pricing and demographic datasets containing:
- Childcare costs by state
- Household income information
- Regional affordability indicators
- Family and population statistics

The datasets were cleaned and transformed to support storytelling-focused analysis and visualization.

---

## Project Objectives
- Analyze childcare cost differences across states
- Identify regions with the highest childcare burden
- Compare childcare expenses relative to income levels
- Visualize affordability trends across the United States
- Communicate insights through data storytelling techniques

---

## Data Preparation
The project included:
- Cleaning missing and inconsistent values
- Standardizing state and regional information
- Aggregating cost-related metrics
- Preparing datasets for geographic visualizations
- Formatting variables for storytelling analysis

---

## Exploratory Data Analysis
Several visualization techniques were used throughout the project, including:
- Choropleth maps
- Heatmaps
- Bar charts
- Trend comparisons
- Regional affordability visualizations

The visualizations focused on highlighting economic differences and the impact of childcare costs on families.

---

## Key Findings
- Childcare costs varied significantly across states
- Several regions showed disproportionately high childcare expenses relative to household income
- Geographic trends revealed affordability challenges concentrated in certain areas
- Visual storytelling techniques helped communicate the financial burden faced by families
