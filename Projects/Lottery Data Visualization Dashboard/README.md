# Lottery Data Visualization Dashboard

## Overview
This project explores lottery datasets using data visualization and dashboarding techniques to analyze number distributions, drawing patterns, and lottery trends. The goal of the project was to transform raw lottery data into interactive and meaningful visual insights using Power BI and Python-based analytics.

The analysis focused on identifying trends and patterns across multiple lottery games through statistical visualization and exploratory data analysis.

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI
- Jupyter Notebook

---

## Data Sources
The project uses lottery datasets containing historical drawing information from games such as:
- Powerball
- Mega Millions
- Pick 10

The datasets included:
- Winning numbers
- Drawing dates
- Frequency counts
- Number distributions

The data was cleaned and transformed for visualization and statistical analysis.

---

## Project Objectives
- Analyze lottery number frequency patterns
- Explore drawing distributions and trends
- Create dashboard-style visualizations
- Compare patterns across multiple lottery games
- Apply statistical visualization techniques to large datasets

---

## Data Preparation
The project included:
- Cleaning and formatting lottery datasets
- Handling missing and duplicate values
- Aggregating number frequency counts
- Preparing data for dashboard visualizations
- Structuring data for multi-chart analysis

---

## Exploratory Data Analysis
Several visualization techniques were used throughout the project, including:
- Scatterplots
- Bubble charts
- Histograms
- Density plots
- Small multiples

The analysis focused on understanding how lottery numbers are distributed over time and identifying recurring trends within the datasets.

---

## Key Findings
- Certain numbers appeared more frequently across historical draws
- Distribution patterns varied between lottery games
- Visualizations highlighted clustering and frequency trends
- Dashboard visualizations improved interpretability of large drawing datasets
