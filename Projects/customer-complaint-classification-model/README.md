# Customer Complaint Classification Model

## Overview
This project focuses on predicting customer complaint resolution outcomes using machine learning techniques and natural language processing (NLP). The analysis uses customer complaint data to classify whether complaints are likely to be resolved promptly based on complaint narratives and related features.

The project applies text preprocessing, feature engineering, classification modeling, and evaluation metrics to analyze customer service and complaint resolution trends.

## Data Sources
The project uses customer complaint datasets containing:
- Complaint narratives
- Product categories
- Consumer issue descriptions
- Resolution outcomes
- Submission information

The dataset was cleaned and transformed to support machine learning classification analysis.

## Key Findings
- Logistic Regression achieved strong classification performance with ROC-AUC around 0.77
- Text preprocessing significantly improved predictive performance
- Class imbalance impacted prediction behavior and required balancing techniques
- Complaint narratives contained meaningful patterns related to resolution outcomes
