# Netflix Content Analysis

## Overview
This project analyzes Netflix content data to explore trends in movies and television shows available on the platform. The analysis focuses on understanding content distribution, genre popularity, release patterns, and viewing trends using exploratory data analysis and visualization techniques.

The project combines multiple Netflix datasets to identify patterns in content performance and platform trends.

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

---

## Data Sources
The project uses Netflix datasets containing:
- Movie and TV show titles
- Genres and categories
- Release years
- Viewing statistics
- Popularity rankings
- Weekly top content information

The datasets were cleaned, merged, and transformed to support exploratory analysis and visualization.

---

## Project Objectives
- Analyze trends in Netflix movies and television content
- Explore genre popularity and viewing patterns
- Identify content distribution across categories
- Visualize trends in streaming performance
- Compare viewing behavior across different content types

---

## Data Preparation
The project included:
- Cleaning missing values
- Standardizing titles and categories
- Merging multiple Netflix datasets
- Formatting date variables
- Aggregating viewing statistics
- Preparing data for visualization and trend analysis

---

## Exploratory Data Analysis
Several visualization techniques were used throughout the project, including:
- Bar charts
- Trend analysis
- Category comparisons
- Viewing distribution visualizations
- Popularity ranking analysis

The analysis focused on identifying patterns in audience engagement and streaming content trends.

---

## Key Findings
- Certain genres consistently dominated viewing trends
- Netflix content distribution varied significantly across categories
- Popular titles demonstrated strong viewing concentration patterns
- Streaming trends changed over time depending on content releases and categories
